# uzgets_backend

Uzgets — Express API, Telegram bot (`token.js`), SMS/to‘lov eshituvchi (`balanceChecker.js`).

## Sozlamalar

`.env` ichida `DATABASE_URL`, `BOT_TOKEN`, `PORT` (masalan `7001`), `WEBAPP_URL`, `INTERNAL_API_SECRET` va boshqalar.

## Ishga tushirish

```bash
npm install
npm start
```

Mahalliy rivojlantirish uchun: `NODE_ENV` ni `development` qilib qo‘yish kerak bo‘lishi mumkin (CORS / localhost).
